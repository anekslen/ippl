template<typename MemorySpace = Kokkos::DefaultExecutionSpace::memory_space, typename T = char>
buffer_type<MemorySpace> ippl::Communicate::getBuffer ( int  id,
size_type  size,
double  overallocation = 1.0 
)

Obtain a buffer of at least the requested size that is associated with the given ID, overallocating memory for the buffer if it's new

Template Parameters
MemorySpaceThe Kokkos memory space for which to allocate the buffer
TThe datatype to be stored in the buffer; in particular, the size is scaled by the size of the data type (default char for when the size is already in bytes)
Parameters
idThe numerical ID with which the buffer is associated (allows buffer reuse)
sizeThe minimum size of the buffer, measured in number of elements of the provided datatype (if the size is in bytes, the default type char should be used)
overallocationThe factor by which memory for the buffer should be overallocated; only used if the buffer with the given ID has not been allocated before; by default, the larger value between 1 and the defaultOveralloc_m member is used
Returns
A shared pointer to the buffer with the requested properties
